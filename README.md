Repository ini berisi tugas akhir dari praktikum Percobaan judul 4 yaitu PHP & Manipulasi Form, yang bertujuan untuk memahami cara kerja pengolahan data menggunakan form pada PHP.

- Tujuan Pembelajaran
Memahami mekanisme dasar form HTML.
Mempelajari cara mengirim data menggunakan method GET & POST.
Mengolah data form menggunakan PHP.
Menerapkan validasi sederhana.
Menampilkan hasil input ke halaman lain atau halaman yang sama.
Melatih penggunaan server lokal (XAMPP).

- Teknologi yang Digunakan
PHP
HTML & CSS
XAMPP / Apache
